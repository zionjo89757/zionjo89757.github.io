npm常用指令

  • npm -v 查看 npm 的版本
  • npm version 查看所有模块的版本
  • npm search package 搜索包
  • npm install package 安装包
  • npm remove package 删除包
  • npm install package –save 并添加依赖
  • npm init 初始化工作空间，创建package.json 文件
  • npm install 下载当前项目所依赖的包
  • npm install package -g 全局安装（一般为工具）

Node模块化实质

全局对象

  • node中有一个全局对象global，它的作用和网页中的window类似
    • 在全局中创建的变量会被作为global的属性保存
    • 在全局中创建的函数会被作为global的方法保存

默认闭包

实际上模块中的代码都包装在一个函数中执行的，并且在函数执行时传入5个实参

  • exports： 该对象用来将变量暴露到外部
  • require： 函数，用来引入外部的模块
  • module： 当前模块本身，exports为当前模块的属性
  • __filename： 当前模块的完整路径
  • __direname： 当前模块所在文件夹的完整路径

引入其他模块

  • 在node中，通过require()函数来引入外部的模块
    • require()可以传递一个文件的路径作为参数，node将会自动根据该路径来引入外部模块。
  • 使用require()引入模块以后，该函数会返回一个对象，这个对象代表的是引入的模块
  • 模块标识
    • 核心模块：由node引擎提供的模板
    • 文件模块：由用户自己创建的模板
